사이트에 접속하기

OVERVIEW

다양한 식물들을 판매하는 사이트를 리뉴얼하여 기존사이트의 문제점을 개선하고 컨텐츠가 잘 드러나도록 기획하였습니다.

As is

  • 메뉴가 너무 많아 헷갈려요.
  • 상품이 빼곡히 붙어있어 답답하고 잘 보이지 않아요.
  • 상품에 관한 정보를 더 알고싶어요.

To be

  • 중복되는 메뉴와 우선순위가 낮은 메뉴들은 없애고 주 카테고리들을 상단에 배치
  • 상품 섹션을 크게 만들고 톤앤매너에 맞게 배치
  • 상품카드의 식물사진을 크게 배치하고 평점 및 가격이 잘 보일수있도록 배치, 바로 장바구니에 넣을수 있게 아이콘 추가

기존 사이트

WIREFRAME

식물을 구매하고자하는 고객에 맞추어 주 카테고리를 상단에 위치시키고 상품위주섹션들로 구성하였습니다.

네비게이션바를 장바구니와 상단이동버튼만으로 구성하여 간소화시켰습니다.

깔끔한 배열의 상품카드를 구상하였습니다.

사용자가 쇼핑중 생동감을 느낄수있도록 슬라이드와 커버플로우 형식의 상품섹션을 구성하였습니다.

할인품목들 사이에 고급상품 섹션을 넣어 감상할 수 있도록 하였습니다.

식물사이트와 밀접하게 식물색과 유사한 색으로 브랜드 컬러를 선정하였습니다.
사용자가 식물이 가득한 식물원이나 숲속에 온 느낌을 느낄 수 있도록 컴포넌트에 색을 사용하였습니다.

COMPONENT

컬러를 적절히 조절하여 톤앤매너에 맞는 색상과 배치로 상품의
관한 정보의 가독성과 심미성의 조화를 맞추었습니다.

MAIN PAGE

주로 사용되는 카테고리들을 상단에 배치하였습니다.
어떤 카테고리인지 잘 보이도록 명확하게 사진을 크게 사용였고 이벤트배너도
크게 사용하여 조화롭게 구성하였습니다.

흰 바탕에 상단왼쪽, 하단오른쪽 나뭇가지 이미지로
깔끔한 자연의 느낌속에 상품들을 배치하였습니다.

슬라이드 형식의 상품카드로 저렴한 가격의 상품을 이동시키며
구경할 수 있도록 구성하였습니다. 움직이는 카드들이 생동감을 느끼게 해줍니다.

커버플로우 형식의 상품카드로 고급식물을 설명과 함께 넘기며
구경할 수 있도록 구성하였습니다.

오늘의 상품들로 꽃색상 배경과 함께 주목되는
느낌으로 상품을 볼 수 있게 배치되었습니다.

사이트에 대한 정보와 공식 블로그로 이동하는
버튼으로 이루어진 간결한 푸터입니다.
